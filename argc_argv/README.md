*************** ARGC and ARGV ****************
* ARGC :				     *
*  --> The number of command line arguments  *
*  --> The size of the argv array            *
*					     *
* ARGV :                                     *
*  --> An array containing the program       *
*      command line arguments.               *
*  --> An array of size argc.                *
*					     *
